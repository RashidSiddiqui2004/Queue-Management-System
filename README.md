# Queue Management System

This is a queue management system built using React, Vite, and Cypress. It allows users to manage their queue and provides a user-friendly interface for interacting with the system.

## Features

- Queue management
- Real-time updates
- Responsive design
- User-friendly interface 

## Installation

To install the dependencies, run the following command:

```bash
npm install
```

## Usage

To start the development server, run the following command:

```bash
npm run dev
```

## Testing

Cypress is used for end-to-end testing. To run the test, run the following command:

```bash
npx cypress open
```

## Deployment

To deploy the application, run the following command:

```bash
npm run build
```
